## Cara Menjalankan Sistem

1. Clone repository

```bash
git clone https://github.com/zakyfawwaz/UAS-PKL_23215031_Zaky.git
```

2. Masuk ke folder project

```bash
cd UAS-PKL_23215031_Zaky
```

3. Install dependency

```bash
composer install
```

4. Salin file environment

```bash
cp .env.example .env
```

5. Generate application key

```bash
php artisan key:generate
```

6. Konfigurasi database PostgreSQL pada file `.env`

7. Jalankan migration

```bash
php artisan migrate
```

8. Jalankan server Laravel

```bash
php artisan serve
```
